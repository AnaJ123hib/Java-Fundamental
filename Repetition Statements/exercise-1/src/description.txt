Write a Java program that implements the following game:
• The program will select a random number between 0 and 100 and the player must guess it.
• In each attempt the player will propose a solution and will be informed if the number
to be guessed is lower or higher than the one proposed.
• The game ends when the number is guessed correctly or a maximum of 12 attempts have been
made, in which case the player will be shown the score obtained according to the following
information:

•It was pure luck! !1-3
•You are good! 4-6
•It's not bad 7
•It can be improved 8
•You look so bad!!! >8
